Projeto de código 2 - escreva código que valida a entrada de cadeia de caracteres
Aqui estão as condições que seu segundo projeto de codificação deve implementar:

Sua solução deve incluir uma do-while ou while iteração.

Antes do bloco de iteração: sua solução deve usar uma Console.WriteLine() instrução para solicitar ao usuário um dos três nomes de função: Administrador, Gerente ou Usuário.

Dentro do bloco de iteração:

Sua solução deve usar uma Console.ReadLine() instrução para obter informações do usuário.
Sua solução deve garantir que o valor inserido corresponda a uma das três opções de função.
Sua solução deve usar o Trim() método no valor de entrada para ignorar caracteres de espaço à esquerda e à direita.
Sua solução deve usar o método no valor de entrada para ignorar maiúsculas ToLower() e minúsculas.
Se o valor inserido não corresponder a uma das opções de função, seu código deverá usar uma instrução para solicitar ao usuário uma Console.WriteLine() entrada válida.
Abaixo (depois) do bloco de código de iteração: Sua solução deve usar uma Console.WriteLine() instrução para informar ao usuário que seu valor de entrada foi aceito.

Comente todo o código no painel Editor de Códigos do Visual Studio

Selecione todas as linhas de código no editor de código
No menu Editar, selecione Alternar Bloquear Comentário.
Escreva o código que implementa cada condição para o projeto de código 2.

Execute seu aplicativo e verifique se seu código valida a entrada do usuário com base nos requisitos especificados.

Por exemplo, quando você executa seu aplicativo, ele deve rejeitar um valor de entrada como "Admin", mas deve aceitar um valor de entrada de " administrator ".

A saída do console para este exemplo deve ser semelhante à seguinte:

Enter your role name (Administrator, Manager, or User)
Admin
The role name that you entered, "Admin" is not valid. Enter your role name (Administrator, Manager, or User)
   Administrator
Your input value (Administrator) has been accepted.
