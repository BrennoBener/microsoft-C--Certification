Projeto de código 1 - escreva código que valida a entrada de números inteiros
Aqui estão as condições que seu primeiro projeto de codificação deve implementar:

Sua solução deve incluir uma do-while ou while iteração.

Antes do bloco de iteração: sua solução deve usar uma Console.WriteLine() instrução para solicitar ao usuário um valor inteiro entre 5 e 10.

Dentro do bloco de iteração:

Sua solução deve usar uma Console.ReadLine() instrução para obter informações do usuário.
Sua solução deve garantir que a entrada seja uma representação válida de um inteiro.
Se o valor inteiro não estiver entre 5 e 10, seu código deverá usar uma Console.WriteLine() instrução para solicitar ao usuário um valor inteiro entre 5 e 10.
Sua solução deve garantir que o valor inteiro esteja entre 5 e 10 antes de sair da iteração.
Abaixo (depois) do bloco de código de iteração: sua solução deve usar uma Console.WriteLine() instrução para informar ao usuário que seu valor de entrada foi aceito.

Verifique se você tem um arquivo .cs programa vazio aberto no Visual Studio Code.

Se necessário, abra o Visual Studio Code e conclua as seguintes etapas para preparar um arquivo .cs programa no Editor:

No menu Arquivo, selecione Abrir pasta.

Use a caixa de diálogo Abrir pasta para navegar até a pasta CsharpProjects e abri-la.

No painel Visual Studio Code EXPLORER, selecione Program.cs.

No menu Seleção de código do Visual Studio, selecione Selecionar tudo e pressione a tecla Delete.

Escreva o código que implementa cada condição para o projeto de código 1.

Execute seu aplicativo e verifique se seu código valida a entrada do usuário com base nos requisitos especificados.

Por exemplo, quando você executa seu aplicativo, ele deve rejeitar valores de entrada como "dois" e "2", mas deve aceitar um valor de entrada de "7".

O exemplo descrito acima, a saída do console deve ser semelhante à seguinte:

Enter an integer value between 5 and 10
two
Sorry, you entered an invalid number, please try again
2
You entered 2. Please enter a number between 5 and 10.
7
Your input value (7) has been accepted.
